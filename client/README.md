# AI Portfolio Generator Frontend

Beautiful React frontend for the portfolio generator.

## Quick Start

```powershell
cd client
npm install
npm run dev
```

The frontend will run on `http://localhost:5173` and proxy API requests to the backend on port 3000.

## Usage

1. Start the backend server (in root directory):
   ```powershell
   node server.js
   ```

2. Start the frontend dev server (in client directory):
   ```powershell
   npm run dev
   ```

3. Open http://localhost:5173 in your browser
